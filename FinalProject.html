Anthony Commendatore & Chris Forte Final Project

For my final project my partner Chris and I created an air hockey game using Khan Academy. This is a single player user versus CPU style game up to five.
On the homescreen of our game there is an instructions button that will take you to another page explaining the game and its rules. This game is very fun and enyjoyable,
it is played with the arrow keys and there are goal horns played every time you or the CPU scores. Check it out! .
